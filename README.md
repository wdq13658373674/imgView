# myPlugin

简单的div内鼠标拖动大图查看全景插件

使用方法：

$('#imageView').imageView({

            width: 1122,//容器宽度
            
            height:700,//容器高度
            
            cursor:'move',//鼠标按下时的显示cursor
            
            imgwidth:1300,//图片宽度
            
            imgheight:1900//图片高度
            
        });
        
兼容性

兼容各个主流浏览器、ie9+
